Git init--------------初始化本地Git仓库

Git add--------------添加文件

Git status--------------查看状态

Git commit--------------提交

Git commit -m "描述"--------------提交

Git push--------------推送到仓库

Git pull--------------从远程仓库拉取数据

Git clone--------------从远程仓库拷贝数据

echo test>文件--------------创建文件（window系统下用）

touch--------------创建文件（Linux系统下用）

echo test>.gitignore--------------忽略上传（使用本条命令会创建一个.gitignore文件，在本文件中添加不需要上传的文件名称保存即可）

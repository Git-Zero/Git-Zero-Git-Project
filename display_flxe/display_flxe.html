display:Flex伸缩容器 - Flex Container 定义在Container

伸缩容器 - Flex Container下面的容器都是伸缩项 - Flex - Item

伸缩项默认生成主轴（main axis）和侧轴（cross axis），主轴从左往右，侧轴从上往下

Flex-Direction伸缩容器改变主轴的方向 定义在Container

  • Row(default)Flex-Direction: Row主轴-水平从左到右排列 1，2，3
  • Row-ReverseFlex-Direction: Row-Reverse主轴-水平从右到左排列 1，2，3
  • columnFlex-Direction: column主轴-垂直从上至下排列 1，2，3
  • column-ReverseFlex-Direction: column-Reverse主轴-垂直从下至上排列 1，2，3

Flex-Wrap伸缩容器改变纵轴方向，根据主轴的方向来改变

  • Nowrarp(default)
  • Wrap默认侧轴从上至下
  • Wrap-Reverse反转 从下至上

Order伸缩项改变指定伸缩项的位置（下标从0开始）定义在伸缩项 - Flex - Item

Justify-content伸缩容器控制伸缩项在主轴的展现方式 定义在Container

  • Flex-Start(default)元素在主轴（页面）上由左或者上开始排列
  • Flex-end元素在主轴（页面）上由右或者下开始排列
  • center元素在主轴（页面）上居中排列
  • space-between元素在主轴（页面）上左右两端或者上下两端开始排列
  • space-around每个元素两侧的间隔相等。所以，元素之间的间隔比元素与边框的间隔大一倍。

Aligin-items伸缩容器控制伸缩项在侧轴的展现方式 定义在Container

  • flex-start弹性盒子元素的侧轴（纵轴）起始位置的边界紧靠住该行的侧轴起始边界（靠上对齐）。
  • flex-end弹性盒子元素的侧轴（纵轴）起始位置的边界紧靠住该行的侧轴结束边界。（靠下对齐）
  • center弹性盒子元素在该行的侧轴（纵轴）上居中放置。（居中对齐）
  • baseline如弹性盒子元素的行内轴与侧轴为同一条，则该值与'flex-start'等效。其它情况下，该值将参与基线对齐。（靠上对齐）

Align-content伸缩容器控制伸缩项所有组成的行（列）在侧轴的展现方式 定义在Container

  • flex-start打包到容器开头的行
  • flex-end包装在容器末端的行
  • center打包到容器中心的行
  • space-between线均匀分布；第一行在容器的开头，而最后一行在容器的结尾
  • space-around线均匀分布，它们之间的距离相等
  • stretch线拉伸以占用剩余空间

flex-grow 伸缩项控制伸缩项在伸缩行的伸展程度 定义在伸缩项 - Flex - Item

flex-grow: integer

flex-shrink伸缩项控制伸缩项在伸缩行的收缩程度 定义在伸缩项 - Flex - Item

flex-shrin: integer
